# Welcome Home

This started as a modification of [Refuge Restrooms](https://github.com/RefugeRestrooms/refugerestrooms) on October 25th, 2014.

## Tech

* Ruby Version - ruby-2.0.0-p247
* Ruby on Rails
* RSpec
* Javascript
* HTML / SASS
* Postgres
* Geocoder Gem
* Google Maps API
* Twitter Boostrap Framework
* Deployed on Heroku
